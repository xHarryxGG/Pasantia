# Carpeta de Manuales de Usuario

Coloca aquí los archivos PDF de los manuales del sistema SIPF.

## Archivos requeridos

| Archivo                     | Descripción                                          | Visible para           |
|-----------------------------|------------------------------------------------------|------------------------|
| `manual-usuario.pdf`        | Manual de instrucciones para usuarios normales       | Usuarios por dpto.     |
| `manual-administrador.pdf`  | Manual completo con funciones de gestión avanzadas   | Administradores        |

## Acceso en la aplicación

Los botones flotantes (FAB) en la esquina inferior izquierda de las pantallas de inicio
abrirán automáticamente estos archivos en una nueva pestaña del navegador.

> ⚠️ Los archivos deben estar presentes en esta carpeta para que el enlace funcione.
> Si no están, el navegador mostrará un error 404.
